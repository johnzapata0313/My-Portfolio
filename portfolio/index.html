The NBA Player Randomizer is a simple web app that gives you a random NBA player along with their team, position, and where they came from before entering the league — whether that be college, high school, or country. When the user clicks the “Get Player” button, the app fetches data from the BallDon’tLie API and then uses the Leaflet API to find and display that location on an interactive map. The project is built using HTML, CSS, and JavaScript, with a clean and centered layout that features a background of an NBA court!
